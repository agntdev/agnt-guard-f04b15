# agnt Guard

Telegram group moderation bot that greets and verifies new members with 60s math captchas, auto-enforces anti-spam rules, logs actions, and provides admin reports.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
